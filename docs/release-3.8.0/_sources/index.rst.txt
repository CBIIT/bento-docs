===========================
Bento Project Documentation
===========================

Welcome! This is the documentation site for the Bento Project. It describes the Bento project's background and explains how to set up a data sharing platform using the Bento framework.

Bento is a software framework being built to support NCIâ€™s `Cancer Research Data Commons <https://datascience.cancer.gov/data-commons>`_ projects, including `Integrated Canine Data Commons <https://datacommons.cancer.gov/repository/integrated-canine-data-commons>`_ (`ICDC <https://caninecommons.cancer.gov/#/>`_), and `Clinical Trial Data Commons <https://datacommons.cancer.gov/repository/clinical-trial-data-commons>`_ (CTDC).

This site provides you with the background and instructions you need to set up your own data sharing platform using Bento.
